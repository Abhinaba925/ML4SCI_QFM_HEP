# Quantum Foundation Model for High Energy Physics

This repository contains the solutions, code implementations, and theoretical write-ups for the Quantum Foundation Model for High Energy Physics assignment. 

## 📂 Repository Contents

* **`Assignment_Quantum_Foundation_HEP.ipynb` (Practical Implementation):**
    A Jupyter Notebook containing quantum circuit implementations using **PennyLane**. 
* **`TaskIII.pdf`:**
    A comprehensive research commentary detailing the mathematical scaffolding of QML. Key topics include:
    * Parameterized Quantum Circuits (PQCs) and exact analytical gradients via the Parameter-Shift Rule.
    * Foundational algorithms: Variational Quantum Eigensolver (VQE) and Quantum Approximate Optimization Algorithm (QAOA).
    * Advanced architectures: Quantum Graph Neural Networks (QGNNs), QCNNs, and Quantum GANs.
    * Trainability challenges in the NISQ era, specifically the Barren Plateau problem and the proposed solution of Geometric/Symmetry-Aware QML.

## 🛠️ Technologies & Frameworks

* **Programming & Typesetting:** Python 3.x, LaTeX
* **Quantum Frameworks:** PennyLane 
* **Libraries:** NumPy, Jupyter
